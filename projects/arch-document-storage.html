Features

Arch Document Storage

Arch Document Storage is a Document Management System for Arch Retail Systems. Users can upload scanned receiving documentation (including claim documentation) to a central server for head office employees and suppliers to access. The application also has a few reports regarding document statistics and user login statistics.

Create Multiple Campaigns

Create more than one campaign at a time and let users vote one each of these campaigns simultaneuosly. Setting up a campaign is easy. Give it a name, a start and end date and just add the campaign candidates.
